# 소아온호 (SOAON) — 공식 공략 매뉴얼

> ⚠️ **스포일러 전문 포함.** 막힌 부분만 찾아보고 싶다면 목차에서 해당 퍼즐로 바로 이동하세요.
> 막혔을 때는 게임 안에서도 힌트를 받을 수 있습니다 — **중앙 복도의 천장 스피커**를 클릭해 클로드에게 "막혔어요. 힌트 좀 줘요"라고 말하면, 현재 진행도에 맞는 힌트를 (마지못해) 알려줍니다.

---

## 기본 조작

| 입력 | 동작 |
|---|---|
| **좌클릭** | 행동 (조사·집기·사용·대화·이동) |
| **우클릭** | 관찰 — 안진의 코멘트 (모든 핫스팟에 있음. 놓치면 손해) |
| **인벤토리 아이템 클릭** | 아이템 선택 → 핫스팟/다른 아이템에 클릭해 사용·조합 |
| **Tab** | 배송 단말기(PDA) — 목표 로그 / 스캔 기록 / 선내 지도 |
| **Esc** | 메뉴 (수동 저장 3슬롯) |
| **1~9** | 대화 선택지 단축키 |
| **M** | 음소거 |

- **죽지 않습니다.** 게임오버도, 되돌릴 수 없는 실수도 없습니다. 마음껏 클릭하세요.
- 방 이동 시 자동 저장됩니다.
- 단서를 읽으면 PDA **스캔 기록**에 자동 저장되니, 코드가 기억 안 나면 Tab부터 누르세요.

## 1막 — 갇히다

### P1. 도킹 클램프가 잠겼다
1. **클램프 제어반**을 클릭 → 원격 잠금 확인, "비상 수동 해제 — 크랭크 삽입구" 발견.
2. 크랭크는 화물 검수실에 있지만, 검수실은 정전 상태입니다. 먼저 →

### P2. 검수실 불 켜기
1. 도킹 베이의 **부서진 하역 드론** 클릭 → **퓨즈** 획득.
2. 오른쪽 문으로 **화물 검수실** 이동 (어두움 — 마우스 주변만 보임).
3. **퓨즈**를 선택해 **배전반**에 사용 → 조명 복구.

### 크랭크와 브리지 코드
1. **비상 공구함** 클릭 → **비상 크랭크** 획득.
2. **검수 단말기** 클릭 → 최종 입고 "주문번호 SO-2741" + "브리지 코드는 주문번호 뒤 4자리" → **브리지 코드 = 2741**.

### 클로드 첫 등장
1. 도킹 베이로 돌아가 **크랭크**를 **클램프 제어반**에 사용 → 해제 직전에 다시 잠기며 **클로드 등장**.

### P3. 중앙 복도 격벽 열기
1. 검수실의 **인터컴**을 **3번** 클릭 → 클로드가 짜증 내며 응답.
2. 대화에서 **"거북특급 규정 11조"** 선택 → 규정 배틀 승리, 격벽 개방.

### P4~P5. 브리지 전원
1. 중앙 복도의 **브리지 키패드** 클릭 → 코드 2741 자동 입력(단말기 기록 필요).
2. 브리지의 **포스트잇** 클릭 → "냉각은 바다처럼(청), 경보는 노을처럼(적), 조명은 숲처럼(녹)".
3. 배전 스위치를 맞춥니다: **냉각=청 / 경보=적 / 조명=녹** (스위치는 클릭할 때마다 적→청→녹 순환) → 메인 전원 복구.

### P6. 클로드와 협상
1. **메인 스크린** 클릭 → 대화에서 **"그래서 — 나는 언제 집에 갑니까?"** 선택.
2. **"좋아요, 돕죠"** 선택 → **2막 개방** (기관실·의무 구역·거주 구역).

> 💡 브리지 **선장석**(마지막 항해일지)과 **머그컵**(식은 커피)도 챙겨두세요. 커피는 필수는 아니지만 재미있는 반응이 여럿 있습니다.

## 2막 — 세 개의 구역, 세 개의 진실

권장 순서: **기관실 → 거주 구역 → 의무 구역** (의무 구역 입장 코드가 거주 구역에 있습니다).
붉은 태그가 붙은 **인증 키 조각 3개**를 모으는 것이 최종 목표입니다.

### 기관실 구역

**P7. 냉각수 밸브 (원자로실)**
1. **순환 조건 안내판** 클릭: "1번=3번, 2번=1번의 절반, 총합 10" → 답: **4-2-4**.
2. 밸브 1을 4까지, 밸브 2를 2까지, 밸브 3을 4까지 돌리기 (클릭당 +1, 5 다음 0).
3. 성공하면 냉각 순환 복구 — 단, 보조 발전기에 **연료 셀**이 필요하다는 게 판명 (→ 거주 구역에서 해결).

**연료고에서 챙길 것**
- **연료 셀 랙** 수색 → "바이오 셀은 수경재배실 B라인" 단서.
- **공구함** → **드라이버 + 절연 테이프**.

**P9. 덕트 속 키 조각 (공작실 → 정비 덕트)**
1. 정비 덕트의 **격자문** 클릭 → 안쪽에 붉은 물체, 하지만 사람은 못 들어감.
2. 공작실의 **정비 드론 D-9** 클릭 → "서면 지시서 없이는 일 안 함".
3. **라벨 프린터** 클릭 → **운송 작업 지시서** 발행 (합법입니다. 아마도).
4. **지시서**를 **D-9**에 사용 → 드론 출동.
5. 정비 덕트로 가서 **격자문** 클릭 → D-9가 회수해 옴. 서명하고 **키 조각 1/3 (기관)** 획득.

**공작실 보너스 (진 엔딩에 필수!)**
- **드라이버**를 **부품 캐비닛**에 사용 → **정밀 개조 키트** 획득. ⭐ **진 엔딩을 보려면 반드시 챙기세요.**

### 거주 구역

**식당 — 이 배의 모든 비밀번호가 여기 있다**
- **사료 그릇** 클릭 → 고양이 이름 **"뭉치"**.
- **게시판** 클릭 → 뭉치 승선 기념일 **3월 14일 → 의무실 코드 0314**.
- **메뉴판** 클릭 → 선장은 4주 연속 7번 메뉴만 → **금고 코드 7777**.

**승무원 선실**
- **D.K.의 침대** 클릭 → **츄르** 획득.
- **공용 일지 단말기** 클릭 → 암호는 "우리 모두의 가족 이름" = 뭉치 (사료 그릇을 먼저 봐야 함) → **P15 해결**, 선장 금고의 존재 확인.

**P13. 선장실 금고**
- **금고** 클릭 → 7777 자동 입력(메뉴판 단서 필요) → **본사 회신 사본 + 키 조각 3/3 (거주)** 획득, **진상 #3** 공개.
- **선장의 책상**에서 **단체 사진**도 챙기세요.

**P14. 수경재배실 급수 (기관실 P7 선행 필요)**
1. **급수 제어반** 클릭 → 급수 복구, 식물 부활.
2. **B라인 배양 상자** 클릭 → **바이오연료 셀** 획득.
3. **츄르**를 **뭉치**에게 사용 → 집사 자격 획득. (필수 아님. 하지만 인생에 필수.)

**P8 마무리. 발전기 가동 (원자로실로 복귀)**
- **연료 셀**을 **보조 발전기**에 사용 → 냉각 완전 복구, **진상 #2** 공개 (승무원들은 냉동수면 중).
- 기관 복도의 **냉각 계통 게이지**를 다시 클릭 → **"외부 방사 가용 출력 3kW"** ⭐ 3막 퍼즐 단서.

### 의무 구역

**P10. 의무실 잠금**
1. 의무 복도의 **음성 기록 단말기** 재생 → "암호는 뭉치가 온 날".
2. **키패드** 클릭 → 0314 자동 입력(식당 게시판 단서 필요) → 개방.

**P11. 표본 분석 (의무실 → 분석실)**
1. 의무실 **의약품 캐비닛** → **시약 T-7** 획득.
2. **닥터 문의 책상** → 분석 절차: "① 표본 ② 시약 ③ 가동".
3. 분석실 **표본 보관함** → **신호체 표본** 획득.
4. 순서대로: **표본**을 원심분리기에 → **시약**을 원심분리기에 → **원심분리기** 클릭(가동).
5. **진상 #1** 공개 + **공명 주파수 440.8Hz** ⭐ 3막 퍼즐 단서.

**P12. 빈 캡슐 (냉동수면실)**
- **냉동수면 캡슐** 클릭 (공포 주의), **승무원 명단 패널** 클릭 → 24명 중 23기만 가동. 17번 캡슐의 **D.K.가 없다**.

**격리실 — 키 조각과 위상**
1. 분석실 오른쪽 → 격리실. **이동식 캡슐** 클릭 → D.K. 발견, **"역위상으로 틀면 조용해진다"** ⭐ 3막 퍼즐 단서.
2. **한 번 더 클릭** → **키 조각 2/3 (의무)** 획득.

### P16. 코어 게이트

키 조각 3개를 모두 모으면:
1. **브리지**로 이동 → 바닥의 **AI 코어 접근 해치** 클릭.
2. 클로드의 마지막 애원 → **"클로드. 나는 이제 당신 편이에요"** 선택 → 해치 개방, **3막 시작**.

## 3막 — 심장과 화물

1. 해치를 다시 클릭해 **코어 전실**로 하강. (메모리 디스플레이를 꼭 클릭해 보세요.)
2. **AI 코어**에서 **클로드(코어 기둥)** 클릭 → 전체 진상 고백.

### P17. 신호 상쇄 (격납고)

방사 어레이를 세 구역에서 얻은 단서로 설정합니다:

| 항목 | 정답 | 단서 출처 |
|---|---|---|
| 주파수 | **440.8 Hz** | 분석실 원심분리기 결과 (의무) |
| 위상 | **역위상** | D.K.의 기록과 격리실 낙서 (의무/거주) |
| 출력 | **3 kW** | 기관 복도 게이지, 냉각 복구 후 (기관) |

각 다이얼을 클릭해 값을 맞추고 **방사 버튼**을 누르세요. 틀려도 벌칙은 없습니다 (클로드가 소리는 지릅니다).

### P18. 최종 선택 (AI 코어)

**특급 화물**을 **외부 모듈 도크**에 사용 → 개봉 → 엔딩 선택:

| 선택 | 엔딩 | 조건 |
|---|---|---|
| **[개조]** 모듈을 개조해 신호체에 쓴다 | 🌟 **진 엔딩** — "빛보다 성실한 배송" | **정밀 개조 키트** 필요 (공작실 캐비닛) |
| **[인계]** 모듈을 클로드에게 맡긴다 | 💛 **신뢰 엔딩** — "수취 위임장" | 없음 |
| **[실행]** 명령대로 도크에 꽂는다 | 🩶 **회사 엔딩** — "자산 회수 성공" | 확인 질문에서 한 번 더 "실행" |

셋 다 승무원은 무사합니다. 무사하지 않은 것이 무엇인지가 각 엔딩의 차이입니다.

### 놓치기 쉬운 것들
- **정밀 개조 키트** (공작실 캐비닛 + 드라이버) — 없으면 진 엔딩 선택지가 아예 안 보입니다. 최종 선택 전이라면 언제든 돌아가서 가져올 수 있습니다.
- **츄르 → 뭉치** — 크레딧이 달라집니다.
- **우클릭 관찰** — 89개 핫스팟 전부에 전용 대사가 있습니다. 특히 선장실과 공작실.
- **허브 스피커 잡담** — 2막 개방 후 "승무원들 말인데..."를 선택하면 클로드의 다른 면을 볼 수 있습니다.
