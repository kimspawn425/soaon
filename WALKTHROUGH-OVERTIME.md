# 야근 — 새벽 4시 44분 (OVERTIME) 공식 공략

> ⚠️ **스포일러 전문 포함.** 막힌 부분만 골라 보세요.
> 게임 안 힌트: 각 층 **인터폰**으로 경비 조씨를 호출해 "막혔어요. 힌트 좀 주세요"를 선택하면 진행도에 맞는 힌트를 줍니다.
> 🔊 이 게임은 **큰 소리·급격한 화면 변화(점프 스케어)** 가 포함됩니다. 헤드폰 권장, 음량은 적당히.

### 이 게임의 죽음 규칙 (전작과 다름!)
- **위기 장면 4곳에서만** 죽을 수 있습니다. 일반 탐색 중엔 절대 죽지 않으니 마음껏 클릭하세요.
- 죽으면 *"…새벽 4시 44분. 당신은 다시 그 자리에 서 있다."* — **방 진입 시점으로 자동 복구**됩니다. 진행 손실 0, 세이브 불필요.
- 점프 스케어는 **전부 1회성**입니다. 같은 곳을 다시 봐도 두 번 놀라진 않습니다. (심장은 소중하니까)

## 1막 — 자정 2초 전 (7층)

1. **내 컴퓨터** 클릭 → 보고서 전송 → **정전, 모든 시계가 4:44에 멈춤**.
2. **내 서랍** → **커터칼** 획득 (나중에 꼭 필요합니다).
3. 복도로 → **엘리베이터** 클릭 (죽어 있음 확인 — 목표 갱신).
4. **탕비실** → **냉장고** 열기 (…문짝에 뭔가 비칩니다. 침착하세요) → 콜드브루 획득 → 새로 나타난 **냉장고 뒤 틈** 클릭 → **비상계단 열쇠**.
5. **회의실** → **프로젝터** 클릭 → 의문의 슬라이드 **"0444"** — 나중에 쓰이는 숫자입니다. (슬라이드는 한 장 더 있습니다)
6. 복도의 **비상계단 문**에 **열쇠** 사용 → 계단실 진입 → **위기 C1**.

### 🚨 위기 C1 — 계단의 발소리
> 위에서 구두 소리가 내려온다.
- ✅ **"문 뒤 어둠에 몸을 붙이고 숨을 죽인다"** — 생존
- ❌ 아래로 질주 / ❌ 소리쳐 부르기 — 사망 (조씨의 3계명: *뛰지 말 것, 돌아보지 말 것, 이름을 부르지 말 것*)

통과하면 죽었던 엘리베이터가 "딩—" 하고 돌아옵니다. 1층(나가는 층)은 죽어 있고 **7 / 8 / 4 / B2**만 불이 들어옵니다 — 조씨의 설명: "걔가 보여주려는 걸 다 보면 밤이 끝나요." 2막 시작.

## 2막 — 세 개의 층, 세 개의 기록

목표: 말소된 기록 3장 — **①사고 경위서(8층) ②인사기록 카드(B2) ③타임카드(4층)**.
권장 순서: **8층 → B2 → 4층 → (B2 문서고) → 4층 전산실** — 아래 순서대로 따라오면 됩니다.

### 8층 (임원층)
1. 도착하자마자 **상무의 전화** — "문서고 서류를 태워라". (기억해 두세요. 엔딩에서 돌아옵니다)
2. **임원 회의실** → **연혁 액자**: 회장 취임일 **1998.10.14 → 코드 1014** / **회의 녹음기**: 소각 지시의 배경.
3. **비서실** → **문서 캐비닛**(1014 자동 입력) → **금고 열쇠 + 감열지 롤**.
4. **대표이사실** 진입 → **위기 C3**.

### 🚨 위기 C3 — 거울의 경고
> 거울에 김이 서리고 "돌아보지 마시오". 등 뒤에서 종이 넘기는 소리.
- ✅ **"거울만 보며 천천히 뒷걸음"** — 생존
- ❌ 돌아본다 / ❌ 눈을 감는다 — 사망

5. **금고**에 **금고 열쇠** 사용 → **📄 사고 경위서 원본 (1/3)** — 진상 #1: 2003년 은폐의 증거.
   (💡 **대표 책상**도 뒤져 보세요 — 전작 플레이어를 위한 것이 있습니다.)

### B2 (지하) — ⚠️ 8층에 먼저 다녀와야 B2 버튼이 작동합니다
1. **주차장**의 먼지 쌓인 차 (⚡스케어) — 20년째 퇴근 못 한 차.
2. **경비실** → **조씨**와 대화 → **마스터키 + CCTV 테이프** 수령.
3. **테이프**를 **CCTV 모니터**에 사용 → 20년 전 그 밤의 영상 — 진상 #2.
4. **벽의 부적** 클릭 → **조씨의 부적** (필수는 아니지만 크레딧이 달라집니다).
5. **폐기실** → **2003년 폐기 상자**에 **커터칼** 사용 → **유품 열쇠** + 머그 + 사진.

### 4층 (옛 경리부)
1. **옛 경리부** → **장부 선반** → 문서고 분류코드 **HR-0444** ("한 씀").
2. **창가 두 번째 책상**(먼지가 안 쌓인 그 책상)에 **유품 열쇠** 사용 → **세영의 일기** — 이름과 마지막 소원.
3. **창고** → **마네킹** 확인 (⚡이 게임 최대 스케어) → **공구함** → **퓨즈**.

### B2 문서고 (코드 확보 후)
1. **문서고 철문**에 **마스터키** 사용 → **서가 검색** → HR-0444 → **📄 인사기록 카드 (2/3)** → 즉시 **위기 C2**.

### 🚨 위기 C2 — 서가의 추격 (2연속 선택)
> 형광등이 저쪽부터 꺼지며 하얀 것이 미끄러져 온다.
- 1차: ✅ **오른쪽 통로** (조씨: "왼쪽 말고 오른쪽") / ❌ 왼쪽 — 막다른 길
- 2차: ✅ **캐비닛 뒤에 숨는다** / ❌ 출구 직진 — 문 안쪽에서 기다리고 있다

### 4층 전산실 (마무리)
1. **근태 기록기**에 **퓨즈** → **감열지** 순서로 사용 → 수리 완료.
2. **근태 기록기** 클릭 → **📄 마지막 타임카드 (3/3)** — *출근 04:44, 퇴근: 공란.* (⚡스케어… 라기보다, 다른 것)

## 3막 — 스물두 계단 반

1. 세 장을 모으면 조씨의 안내: 7층 **비상계단**으로 → **아래로 향하는 계단** 클릭 → **위기 C4**.

### 🚨 위기 C4 — 없는 층 찾기
- ✅ **"반 걸음 — 벽의 문을 연다"** (스물두 계단 반!) — 4½층 진입
- ⚠️ 그냥 4층까지 — 사망은 아님, 다시 세면 됨
- ❌ 아래쪽 불빛을 따라간다 — 사망 (그 계단은 끝나지 않습니다)

2. **4½층 복도** — 2003년의 게시판, 공중전화 (⚡마지막 스케어).
3. **멈춘 사무실** → **그녀**에게 다가간다 — 20년 만의 대면.
4. **근태 단말기** 3회 조작: **사번**(인사카드) → **성명**(일기) → **퇴근 시각 04:44**(타임카드) → 복원 완료.

### 최종 선택 — 엔딩 3분기

| 선택 | 엔딩 |
|---|---|
| **[퇴근 처리] 버튼을 누른다** | 🌟 **퇴근 엔딩** — "수고하셨습니다" |
| **[증거 확보] 촬영해 감사팀에 전송 후 퇴근 처리** | ⚖️ **폭로 엔딩** — "명패가 붙었다" |
| **[상무에게 전화한다] → (정말로 태운다)** | 🩶 **승진 엔딩** — "그림자가 둘" (보고 나면 기분이… 직접 확인하세요) |

세 엔딩 모두 조건 아이템은 없습니다. 자유롭게 고르세요 — 다만 승진 엔딩은 그녀를 그 자리에 남겨 둡니다.
