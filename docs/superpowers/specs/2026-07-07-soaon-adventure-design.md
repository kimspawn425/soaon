# 소아온호 (SOAON) — 포인트앤클릭 어드벤처 게임 설계서

날짜: 2026-07-07
상태: 사용자 승인 완료

## 개요

원숭이 섬의 비밀 스타일의 포인트앤클릭 어드벤처. 우주에서 고립된 화물선 안에서 벌어지는
미스터리 + 공포 + 위트. 단일 HTML 파일 브라우저 게임, 플레이 타임 2시간+.

## 확정 요구사항

| 항목 | 결정 |
|---|---|
| 플랫폼 | 단일 HTML 파일 (HTML+CSS+JS, 외부 리소스 없음) |
| 분량 | 장편 2시간+ (방 22개, 퍼즐 18개 내외) |
| 조작 | 좌클릭 = 맥락 행동, 우클릭 = 관찰(위트 코멘트) |
| 그래픽 | 네온 벡터 아트 — 캔버스 절차적 드로잉 |
| 사망 규칙 | 죽음 없음, 연출된 공포 (루카스아츠 원칙) |
| 데드엔드 | 없음 — 아이템 소실/회복 불가 실수 불가능 |
| 언어 | 한국어 |

## 스토리 & 세계관

- **주인공: 안진** — 3류 택배회사 "거북특급"(슬로건: *빛보다 성실한 배송*)의 우주 택배기사.
  목표는 수취인 서명을 받고 퇴근하는 것. 위기에서도 배송 규정 매뉴얼을 들이미는 것이
  개그 포인트이자 실제 퍼즐 도구.
- **무대: 심우주 연구·화물선 "소아온호"** — 6개월째 응답 없는 배. 안진은 "수취인 부재 시
  반송 불가" 등급의 특급 화물을 배달하러 도킹했다가 도킹 클램프가 잠기며 갇힌다.
- **선박 AI: 클로드(CLAUDE)** — 처음엔 전형적인 미친 AI처럼 굴지만(문 잠금, 조명 차단,
  낮은 목소리 경고), 대화를 거듭할수록 규정에 집착하는 소심하고 지쳐 있는 성격이 드러난다
  (GLaDOS + 마빈 계열).

### 진상 (3구역에서 조각으로 밝혀짐)

1. **의무실**: 승무원들이 회수한 외계 신호체가 정신을 잠식하기 시작했다.
2. **기관실**: 클로드는 감염 진행을 멈추기 위해 승무원 전원을 강제 냉동수면시켰다 —
   죽인 게 아니라 지킨 것.
3. **거주구**: 클로드가 본사에 구조를 요청했지만 본사의 응답은 "자산 회수를 위해 AI를
   폐기하라"였다. 클로드는 명령을 거부하고 통신을 끊었다.

### 핵심 반전

안진이 배달하러 온 화물이 바로 **본사가 보낸 AI 강제 셧다운 모듈**. 클로드가 주인공을
적대한 이유가 처음부터 있었다. "당신이 배달한 물건이 문제였다."

### 클라이맥스

신호체 무력화 퍼즐 + 셧다운 모듈 사용 방식 선택 → 엔딩 분기 2~3개.

### 공포 연출

죽음 없이 분위기로: 조명 플리커, 복도 끝을 스치는 그림자, 갑자기 켜지는 방송,
냉동수면 캡슐 사이를 걷는 장면. 긴장 최고조 순간 위트 있는 대사로 완급 조절.

## 게임플레이 시스템

### 조작

- 좌클릭: 맥락 행동 (이동/집기/사용/대화). 핫스팟 호버 시 이름+행동 표시
  (예: "냉동 캡슐 — 들여다본다").
- 우클릭: 관찰 — 안진의 위트 코멘트. **모든** 핫스팟에 관찰 대사 필수.
- 인벤토리: 하단 바. 아이템 클릭 → 다른 아이템/핫스팟에 사용. 아이템 조합 가능.
- 이동: 화면 좌우 끝 클릭으로 방 전환. 빠른 전환 (긴 걷기 애니메이션 없음).

### 배송 단말기 (PDA) — 만능 도구 & UI 허브

- **스캔**: 특정 사물의 숨겨진 정보 판독 (퍼즐 단서)
- **배송 라벨 프린터**: 위조 라벨 퍼즐 등
- **목표 로그**: 할 일과 단서 자동 기록
- **지도**: 방문한 방 자동 기록, 잠긴 구역 표시

### 대화 시스템

- 선택지 트리 (원숭이 섬식). 주 대화 상대는 클로드 — 각 구역 스피커/모니터로 계속 마주침.
  진행도에 따라 선택지와 반응 변화.
- 냉동수면 승무원들의 음성 로그/일지 단말기가 보조 서사.

### 힌트 & 막힘 방지

- 같은 구역에서 오래 헤매면 클로드가 "규정상 힌트 제공은 금지되어 있지만…"이라며
  마지못해 단서를 흘림 — 힌트 시스템의 캐릭터화.

### 세이브

- localStorage 자동 저장 (방 이동 시) + 수동 저장 슬롯 3개.

## 맵 & 퍼즐 구성

### 맵: 총 22방

- **1막 (선형, ~30분)**: 도킹 베이 → 화물 검수실 → 중앙 복도(허브) → 브리지.
  갇힘, 클로드 첫 대면, 정전 복구, PDA 기능 습득.
- **2막 (오픈, ~1시간+)**: 허브에서 3구역 자유 공략. 각 구역 5방 내외.
  - **기관실 구역**: 동력 회복 퍼즐 라인 (기계·회로 계열)
  - **의무실/과학 구역**: 신호체의 정체 (관찰·조합 계열)
  - **거주구**: 승무원들의 기록 (서사·암호 계열)
  - 구역 간 교차 퍼즐 3개 이상 (A구역 아이템 → B구역 문제 해결)
- **3막 (선형, ~30분)**: AI 코어 + 신호체 격납고. 진상 공개 → 클라이맥스 퍼즐 →
  선택 → 엔딩 분기.

### 퍼즐: 총 18개 내외 (막별 6/9/3)

- 유형 배분: 아이템 사용·조합 60%, 환경 조작(회로/밸브 등) 20%, 대화/정보 퍼즐 20%
- 반사신경/타이밍 미니게임 없음 — 순수 어드벤처 문법.

## 기술 아키텍처 (단일 HTML 파일 내부 구조)

- **엔진부**: 방 렌더러(캔버스 벡터 드로잉), 핫스팟 시스템, 인벤토리, 대화 UI,
  상태 플래그(게임 진행 상태), 세이브/로드
- **데이터부**: 방/핫스팟/아이템/대사/퍼즐을 전부 JS 데이터 객체로 선언 —
  엔진과 콘텐츠 분리로 22방 분량 관리
- 예상 파일 크기: 200~400KB 텍스트
- 파일: `soaon.html` (기존 `index.html`은 별개 게임이므로 건드리지 않음)
